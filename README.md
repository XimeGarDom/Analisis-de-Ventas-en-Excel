# Análisis-de-Ventas-en-Excel
El proyecto muestra un informe de ventas utilizando diversas herramientas de Excel. 
Consta de tres hojas que son: 
1. Datos - Ventas: Donde se muestran nuestros datos origen, que fueron la base para la elaboración del informe.
2. TD - Ventas: Donde se muestran tablas y gráficas dinámicas realizadas con los datos, para un análisis inicial de estos.
3. Informe: Se muestra un informe general y dinámico a modo de resumen, el cual puede funcionar como guía para la toma de decisiones.

Algunas de las herramientas utilizadas para la elaboración de este documento son: 
- Formato de Tablas de Excel
- Formatos condicionales de Celdas
- Tablas dinámicas
- Gráficas Dinámicas
- Segmentaciones
- Conexiones de Informes
